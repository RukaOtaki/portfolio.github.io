WORKS

成果物

  • ポートフォリオ

    紫陽花をテーマにした、やさしく透明感のあるポートフォリオサイトです。
    水色を基調とした淡い色合いと柔らかいデザインで、梅雨の静けさと儚さを表現しました。
    コーディングは Visual Studio Code を用い、HTMLとCSSで構築。
    モバイル表示にも対応しています。

  • 黙示録ノ記憶

    日々の出来事や思考を記録・管理できる日記アプリケーションです。
    日記の登録・編集・削除・検索機能を備えており、
    ユーザーが直感的に操作できるようシンプルでわかりやすいUIを設計しました。
    コーディングは Visual Studio Code を用い、HTML/CSS/JavaScriptで構築。
    モバイル表示にも対応しています。

  • 黙示録ノ記憶 LP

    日々の出来事を記録できる日記アプリケーションの紹介用ランディングページです。
    シンプルかつ幻想的な世界観を意識し、アプリの機能や魅力が視覚的に伝わるよう
    構成・デザインしました。
    コーディングは STUDIO を用い構築。モバイル表示にも対応しています。
